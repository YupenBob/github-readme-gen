# GitHub README Generator

Generate or improve GitHub README.md for open source projects using AI-powered pattern analysis.

## Features

- Analyzes project type and applies appropriate README patterns
- Based on research of 13 high-star (10k+) GitHub projects
- Supports multiple project types: frameworks, educational, resource collections, tools

## Quick Start

1. Read `references/patterns.md` for high-star README patterns
2. Read `assets/template.md` for the reusable template
3. Ask user about their project (name, URL, type, features)
4. Generate README.md content

## Project Types

| Type | Style |
|------|-------|
| Framework/Library | Badges, calm, ecosystem table |
| Educational | Story opening, multi-language, Why motivation |
| Resource Collection | High-density lists, no emotion |
| Tool/Command | Quote opening, mission-driven |

## Files

- `SKILL.md` - Main skill definition
- `references/patterns.md` - High-star README research
- `assets/template.md` - Reusable README template
